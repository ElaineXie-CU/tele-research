# 框架一：全球高中低轨卫星发展现状（截至2026年3月）

## 一、LEO低轨星座：群雄并起

### 1. Starlink（SpaceX）——绝对霸主

**规模（截至2026年3月）：**
- **活跃在轨卫星：约10,000颗**（Wikipedia数据：10,020颗；Grokipedia：9,920颗）
- 已发射总量：约**11,600+颗**（含退役/离轨）
- 最新里程碑：2026年3月17日第10,000颗活跃卫星发射，3月20日突破10,000颗
- 下次发射：2026年3月26日第375批次，29颗，届时累计发射将达**11,667颗**

**技术特点：**
- 轨道高度：340km~550km（多层轨道壳）
- 已实现全球大部分区域覆盖（含极地）
- 手机直连卫星（Direct to Cell）：已与T-Mobile合作，2024年开始测试
- Starlink Mini：面向个人便携用户
- Starshield：军用版本，已为美国政府提供服务

**商业进展：**
- 服务国家：100+个
- 用户数：超300万（估计）
- 已进入车载、船舶、航空等多个垂直场景
- Tesla汽车内置Starlink连接

**核心优势：**
- 成本优势：猎鹰9号可回收火箭，大幅降低单星发射成本
- 规模壁垒：先发优势+频轨资源抢占，后来者难以追赶
- 垂直整合：星载相控阵天线自研，终端成本持续下降

---

### 2. Amazon Leo（原Project Kuiper）——追赶者

**重大更新：**
- **2025年11月正式更名**：从"Project Kuiper"更名为"Amazon Leo"
- 2025年12月：与SpaceX签订发射合同，使用猎鹰9号发射（标志着双方从竞争走向合作）
- 星座总规模：**3,236颗卫星**

**FCC许可要求（硬性截止日期）：**
- 2026年7月30日前：必须部署并运营**一半**（1,618颗）
- 2029年7月30日前：完成全部星座
- 发射合同总投入：超**100亿美元**

**当前进度：**
- 发射总数：约**150颗+**（最后一批24颗，共超150颗）
- 预计2026年开始住宅用户服务
- 2026年7月前目标：700颗在轨 → 将超越OneWeb成为全球第二大LEO星座

**与Starlink的差异：**
- 主要目标：政府、企业、偏远地区（vs Starlink的消费市场为主）
- 已获Verizon、AT&T等美国运营商合作意向
- 定位为AWS云服务的网络延伸

---

### 3. Eutelsat OneWeb——欧洲力量

**当前状态：**
- 已实现全球覆盖（2023年6月完成最终星座部署和地面站建设）
- 2025年：与SES的合并完成（合并后为欧洲最大卫星运营商）
- 2025年：与巴西、泰国空客达成合作，**2026年将开启海外商用服务**

**技术参数：**
- 卫星数量：约648颗（第一代OneWeb）
- 轨道高度：1,200km（LEO）
- 特点：与SES合并后，可与MEO星座协同，提供混合服务

---

### 4. Telesat Lightspeed（加拿大）——艰难前行

**最新动态（2026年3月）：**
- 全球商业服务推迟至**2028年**（原计划2025-2026）
- 已与SpaceX签订14次发射合同，每次最多18颗卫星，从2026年年中开始
- 前两颗"探路者"卫星预计**2026年12月**发射
- 批量发射从**2027年年中**开始
- 战略转向：**国防/政府市场优先**（超过100亿美元订单储备）

**技术参数：**
- 预计卫星数量：约298颗
- 轨道高度：LEO
- 定位：面向企业、政府、海事、航空等高端市场

## 二、MEO中轨星座

### SES O3b mPOWER——最成熟的MEO系统

**最新进展（2026年3月）：**
- **第9和第10颗卫星已投入运营**（2026年3月宣布）
- 此前2025年7月发射的一对卫星已于2025年6月/2026年3月陆续投入使用
- 截至2026年，O3b mPOWER已有**10颗**在轨运行
- 与Lynk Global合作：将Lynk的LEO卫星与SES的MEO卫星及地面关口站联通，实现D2D（设备直连卫星）

**特点：**
- 轨道高度：约8,000km（MEO）
- 相比GEO：时延大幅降低（约120ms vs 600ms）
- 相比LEO：单星覆盖范围更大，切换频率低，更适合企业/政府用户
- 定位：高端企业、政府、海事、航空

# 框架二：我国卫星通信发展情况

## 一、ITU频轨资源申请：史无前例的大规模申报

**重大事件（2025年12月25-31日）：**

中国向ITU提交新增**20.3万颗**卫星的频率与轨道资源申请，覆盖**14个卫星星座**，包括中低轨卫星。这是中国迄今规模最大的国际频轨集中申报。

**背景解读：**
- 频轨资源遵循"先到先得"原则，美国（SpaceX）和英国（OneWeb）已抢占大量资源
- 20万颗申请是中国应对全球LEO竞赛的战略性举措
- "无线电创新院"主导申报，中国星网作为共建单位之一

**战略意义：**
- 为未来GW星座扩容抢占频轨资源
- 与SpaceX的"占据资源即占据市场"战略对标
- 支撑"航天强国"国家战略

---

## 二、GW星座：中国的"星链"计划

**规模（规划）：**
- 总计规划发射：**12,992颗**卫星
- GW-A59子星座：6,080颗，500km以下极低轨道
- GW-A2子星座：6,912颗，1,145km近地轨道
- 两个子星座协同，形成全球互联网卫星覆盖

**技术特点：**
- 未来将推出**手机直连卫星**通信模式
- 空天地一体6G互联网计划的重要组成部分
- 我国第一个巨型卫星互联网计划

**建设进展：**
- 2025年12月：成功发射卫星互联网低轨**20组**卫星
- 2026年1月：中国星网启动**200亿元**规模的二代星核心部件招标
- 项目已进入技术协议签署和排产备货阶段
- 预计2026年持续高频发射

# 框架三：联通能做什么——基于现有能力和资质

## 一、联通现有卫星相关资质盘点

### 已获得：卫星移动通信业务经营许可（A13-1）

**获牌时间：2025年9月8日**

> 工信部向中国联合网络通信集团有限公司颁发卫星移动通信业务经营许可，中国联通可依法开展手机直连卫星等业务，深化应急通信、海事通信、偏远地区通信等场景应用。

**许可内容（具体条款）：**
- 手机直连卫星业务
- 应急通信场景应用
- 海事通信场景应用
- 偏远地区通信场景应用

**落地情况（截至2025年9月）：**
- 已在**8个省市**开通服务
- 初期覆盖重点：山区、海洋、沙漠等地面网络盲区

### 联通的其他卫星相关资源

| 资源 | 说明 |
|---|---|
| 5G网络 | 全球规模领先、速率领先的5G共建共享精品网络 |
| 全球海陆缆 | 丰富的国际传输资源 |
| MWC 2026展示 | 联通在MWC 2026上展示了"联通卫星"服务 |
| 6G前瞻布局 | 已在空天地一体化与低空智联网方向布局 |

---

## 二、联通的SWOT分析

### 优势（Strengths）
- ✅ 已获得A13-1卫星移动通信牌照（先发优势）
- ✅ 强大的地面5G网络基础设施
- ✅ 丰富的国际海陆缆资源
- ✅ MWC 2026上已展示"联通卫星"服务品牌
- ✅ 工信部政策支持（指导意见明确鼓励运营商参与）

### 劣势（Weaknesses）
- ❌ 卫星资产几乎为零（无自有卫星，无星座）
- ❌ 卫星通信运营经验不足
- ❌ 卫星终端产业链不完善
- ❌ 规模和资金不如移动、电信

### 机会（Opportunities）
- 🌟 中国GW星座2026年加速组网 → 可作为国内卫星运营商的接入合作方
- 🌟 天通卫星已有成熟终端 → 可快速接入提供服务
- 🌟 手机直连卫星趋势 → 存量手机无需换终端即可接入
- 🌟 应急通信、海事、偏远地区需求旺盛
- 🌟 6G空天地一体化方向明确

### 威胁（Threats）
- ⚠️ 中国电信已拥有天通卫星运营权（先发优势）
- ⚠️ 中国移动资金更雄厚，可能获得更多卫星资质
- ⚠️ SpaceX Starlink在中国落地受政策限制
- ⚠️ 卫星终端成本和生态仍需培育

---

## 三、联通可行的四大方向

### 方向一：卫星通信服务集成商（最可行）

**核心思路：** 不造卫星，做卫星通信能力的整合者和分发者

**具体路径：**
1. **接入天通卫星**：与中国电信天通系统合作，成为天通服务的二级分销商
2. **接入GW星座**：待GW星座商业化后，争取成为其地面网络和用户服务合作方
3. **接入国际卫星**：与Starlink、OneWeb等洽谈，为在华外资企业和政府机构提供跨境卫星服务

**目标场景：**
- 偏远山区/牧区宽带接入
- 海上船只通信（渔船、海上作业平台）
- 应急指挥通信
- 跨国企业分支机构

**优势：** 轻资产运营，无需自建卫星，利用现有网络和用户资源
**挑战：** 需要与已有牌照方合作，协调商业分成

---

### 方向二：应急通信保障服务

**背景：** 卫星通信在地震、洪水等灾害中是唯一可靠的通信手段

**联通已有基础：**
- A13-1许可明确包含应急通信场景
- 已展示"联通卫星"应急通信产品
- 5G应急通信车可与卫星链路结合

**具体产品：**
- 卫星应急通信指挥车
- 融合通信平台（卫星+5G+固网）
- 灾害场景下的临时通信覆盖

**市场：** 政府采购、央企应急体系、地方政府

---

### 方向三：6G空天地一体化先行者

**趋势：** 6G标准明确将卫星纳入一体化网络架构

**联通已有布局：**
- MWC 2026上展示6G前瞻研究方向：空天地一体化、低空智联网
- 已开展相关技术预研

**建议行动：**
- 参与3GPP 6G卫星通信标准制定
- 与航天科技/航天科工合作进行技术验证
- 探索卫星与5G基站的混合组网技术

**战略意义：** 在6G标准中抢占卫星通信话语权

---

### 方向四：卫星终端与芯片合作

**背景：** 手机直连卫星需要终端芯片支持

**机会：**
- 华为Mate系列已支持天通卫星
- 高通、联发科已推出支持卫星通信的SoC
- 卫星通信功能正在从旗舰向中端普及

**联通可做的事：**
- 与手机厂商合作，定制支持联通卫星业务的合约机
- 与芯片厂商合作，推动卫星通信功能的普及
- 打造"联通卫星+5G"融合终端产品

| ⭐⭐ 中 | 卫星转发器频率使用许可 | 与卫星运营商合作 | 中 | 通过合作方式间接获得 |
| ⭐⭐ 中 | 跨境卫星通信服务许可 | 配合"一带一路"战略 | 中 | 有政策和外交需求支撑 |
| ⭐ 低 | 自有卫星星座频轨资源 | ITU申请+国家统筹 | 极高 | 需要国家战略支持，超出运营商能力 |

# 框架五：地面网络与卫星通信的技术关系

## 一、技术演进脉络：从3G到6G，卫星的角色变化

### 3G时代：卫星作为"孤岛"补充

```
用户终端 ←→ 地面基站 ←→ 核心网
    ↕
  卫星
  (独立系统)
```

- **技术关系：** 卫星与地面网络**完全独立**，无标准化接口
- **卫星作用：** 仅作为地面网络无法覆盖区域的"补充覆盖"（海洋、沙漠、极地）
- **代表系统：** Inmarsat、Thuraya（话音+短信为主）
- **关键技术：** 专用卫星终端，与手机不兼容
- **核心局限：** 终端专用、价格昂贵、与地面网络完全割裂

---

### 4G时代：卫星作为"回传"管道

```
用户终端 ←→ 地面基站 ←→ 卫星回传链路 ←→ 核心网
                          ↑
                     卫星只做"传话"
                   卫星不改地面协议
```

- **技术关系：** 卫星开始与地面网络融合，但仅作为**传输管道（回传/Backhaul）**
- **卫星作用：** 将偏远地区基站的数据通过卫星传回核心网
- **代表应用：** 海上钻井平台、沙漠矿区、海岛的4G基站回传
- **关键技术：** VSAT卫星小站 + 地面基站组合
- **核心进步：** 地面用户可用普通4G终端，但服务范围仍依赖基站

---

### 5G时代：卫星成为"接入网"（NTN标准确立）

这是历史上最重要的技术转折——

```
用户终端 ←→ 卫星接入 ←→ 5G核心网
     ↑            ↑
普通5G手机   不再需要基站
可直接连卫星！
```

**3GPP Release 17（2022年）首次引入NTN（Non-Terrestrial Network）：**

| 技术要素 | 内容 |
|---|---|
| **标准名称** | 5G NR-NTN（基于5G新空口的非地面网络） |
| **核心突破** | 手机无需专用终端，通过软件升级即可连接卫星 |
| **适用频段** | sub-2GHz（FR1频段，最高7.125GHz） |
| **覆盖能力** | 单星覆盖半径可达数百公里 |
| **峰值速率** | 下行数十Mbps（所有用户共享） |
| **应用场景** | 应急通信、偏远地区宽带、海上/航空连接 |
| **补充标准** | NB-IoT NTN（卫星物联网，窄带场景） |

**3GPP Release 18（2024年）：进一步增强NTN**

- 提升移动性支持
- 改善功耗效率
- 提升吞吐量
- 扩大覆盖范围

**3GPP Release 19（正在完成，2025-2026年）：**

- 5G-Advanced（5.5G / 5G-A）正式标准
- 高通在MWC 2026发布**X105调制解调器**，支持Release 19并**集成卫星通信**
- Viasat在MWC 2026上演示**3GPP卫星语音通话**（车载场景）
- Fraunhofer IIS参与展示

**这意味着：5G手机直连卫星不再是演示，而是即将商用！**

---

### 5G-Advanced（5G-A / 5.5G）时代：融合加速

**MWC 2026核心动态：**

1. **高通X105调制解调器发布**
   - 支持5G-Advanced，**Release 19就绪**
   - **原生集成卫星通信能力**
   - 预示5G-A不只是"简单更新"，而是迈向6G的平台

2. **ST Engineering iDirect在Satellite 2026大会上演示3GPP NTN**
   - 展示**原生5G NR卫星接入**，无缝对接5G核心网
   - 支持与地面网络及其他3GPP/非3GPP NTN系统互操作
   - 被称为"迈向统一5G NTN部署的重要一步"

3. **Gilat展示5G NTN端到端连接（跨GEO卫星）**
   - 验证了GEO、MEO、LEO全轨道3GPP标准化接入的可行性

**5G-A与卫星的关系：**
- 不是"卫星替代5G"，而是"卫星扩展5G覆盖"
- 手机在有5G基站时用基站，无基站时自动切换卫星（对用户透明）
- 运营商无需建设两套核心网，一套5G核心网管两个接入网

---

### 6G愿景：天地一体，卫星是"原生组成"

```
6G核心网 ←→ 地面基站 + 卫星接入 + 高空平台(HAPS) + Wi-Fi
                    ↓              ↓            ↓
              城市/农村     海洋/偏远地区   临时/应急
```

**3GPP R21（首批6G标准，预计2026年6月批准）：**

- R21将提供**首套6G技术规范**
- 卫星与地面网络**原生融合**将成为6G基础特性，而非可选功能
- 愿景：用户手机在任意位置——城市、农村、海洋、飞机上——都能获得**相同质量的连接**，卫星和地面网络对用户完全透明

**6G卫星融合的核心特征：**

| 特征 | 说明 |
|---|---|
| 多接入融合 | 5G NR + Wi-Fi + 卫星 + HAPS在同一核心网下统一管理 |
| 服务连续性 | 用户从地面基站切换到卫星过程中，业务不中断 |
| 统一空口设计 | 卫星和地面使用相同的无线空口标准（演进自5G NR） |
| 智能化调度 | AI驱动的网络选择，根据位置、网络状态自动选择最优接入 |
| 内生安全 | NTN安全机制（加密、终端鉴权）成为标准内置功能 |

## 二、地面网络与卫星网络的技术对比

| 维度 | 地面5G网络 | LEO卫星 | MEO卫星 | GEO卫星 |
|---|---|---|---|---|
| **时延** | 1-10ms | 20-40ms | ~120ms | ~600ms |
| **覆盖** | 几公里/基站 | 全球（含极地） | 中纬度覆盖 | 1/3地球（星下点） |
| **带宽** | 高（百Mbps-Gbps） | 中（数十Mbps/用户） | 中高 | 低-中 |
| **终端** | 普通手机 | 改装手机（部分支持） | 改装手机（部分支持） | 专用终端 |
| **成本** | 高（基站+光纤） | 中（规模效应后降低） | 高 | 低（单星覆盖大） |
| **部署速度** | 慢（选址/光纤） | 快（批量发星） | 中 | 快（单星即可服务） |
| **适合场景** | 城市/密集区域 | 全球宽带接入 | 企业/政府宽带 | 广播/应急 |

**关键结论：**
- 地面网络和卫星网络是**互补关系**，不是竞争关系
- 5G-A/6G时代，两者将在**同一核心网**下融合运行
- LEO是卫星宽带的未来（低延迟+高带宽）
- GEO/MEO仍将在广播、物联网、应急等场景发挥作用

## 三、卫星通信网络的技术基础

### 1. 轨道基础知识

```
极低轨 (LEO)    200-300km    ISS轨道高度         时延最低，但寿命短
低轨 (LEO)      300-1200km   Starlink主轨道      时延低≈20-40ms
中轨 (MEO)      2000-20000km GPS/北斗/OGPS星座  时延≈120ms
高轨 (GEO)      35786km      通信广播卫星        时延≈600ms
```

### 2. 卫星通信系统组成

```
┌──────────────────────────────────────────────────────┐
│                    卫星通信系统架构                    │
├──────────────────────────────────────────────────────┤
│                                                       │
│  用户终端 ──→  用户侧链路（Uplink/Downlink）          │
│                  ↕                                      │
│              卫星（处理/转发）                          │
│                  ↕                                      │
│  关口站 ──→  馈电链路（Feeder Link）                  │
│                  ↕                                      │
│              地面网络 / 互联网                          │
│                                                       │
└──────────────────────────────────────────────────────┘
```

**各组成部分：**

| 组件 | 功能 | 关键技术 |
|---|---|---|
| **卫星有效载荷** | 接收、放大、转发信号 | 转发器、相控阵天线、星上处理 |
| **用户终端** | 与卫星通信 | 卫星调制解调器、相控阵天线 |
| **关口站（Gateway）** | 连接卫星与地面网络 | 大口径天线、高功率发射 |
| **卫星控制中心** | 轨道控制、姿态管理 | TT&C（遥测遥控） |
| **运营支撑系统** | 用户管理、计费 | 核心网集成 |

### 3. 手机直连卫星的技术路径

**路径一：3GPP NTN（标准化路径，5G-A/6G主流）**
- 手机内置NTN芯片（高通、联发科已推出）
- 通过5G NR空口直接连接卫星
- 无需改装，存量手机可软件升级支持
- **代表：** 华为Mate系列连接天通卫星、苹果iPhone 14连接Globalstar

**路径二：专有协议（传统路径）**
- 卫星使用专有频段和协议（不是5G）
- 需要专用手机或模块
- **代表：** 天通卫星（S波段）、Inmarsat

**路径三：地面基站回传（最成熟）**
- 卫星仅作为基站到核心网的传输链路
- 用户使用普通手机，但基站需要卫星回传
- **代表：** 海上油气平台、沙漠矿区

### 4. 卫星互联网的技术瓶颈

| 瓶颈 | 说明 | 当前解决方案 |
|---|---|---|
| **雨衰（Rain Fade）** | Ka/Ku波段信号被降雨吸收 | 链路余量设计、纠错编码 |
| **星蚀（Eclipse）** | 卫星进入地球阴影，无太阳能 | 蓄电池+地面供电 |
| **时延抖动** | LEO高速移动导致时延变化 | TCP加速、星上处理 |
| **终端天线** | 高增益定向天线体积大 | 相控阵天线（电调，无需机械转动） |
| **频率干扰** | 与地面5G共用频段可能干扰 | 空间隔离、功率控制 |

### 5. 卫星与地面5G核心网的融合架构

```
            ┌─────────────────────────────┐
            │        5G/6G核心网          │
            │   （统一控制平面+用户平面）   │
            └──────────┬──────────────────┘
                       │
         ┌─────────────┼─────────────┐
         │             │             │
    ┌────▼────┐  ┌─────▼────┐  ┌─────▼────┐
    │地面基站 │  │卫星接入网关│  │ Wi-Fi AP │
    │ (gNB)  │  │(Sat-GW)  │  │          │
    └────┬────┘  └─────┬────┘  └──────────┘
         │             │
         ↓             ↓
    用户在市区      用户在海上/偏远地区
    (地面接入)      (卫星接入)
```

**融合核心网的关键技术：**
- **统一鉴权：** 用户无论通过哪种方式接入，都用同一套SIM卡/鉴权
- **无缝切换：** 5G LAN技术实现地面↔卫星的业务连续性
- **统一计费：** 按实际使用量计费，接入方式对用户透明

# 总结与建议

## 核心判断

1. **全球卫星互联网格局已基本成型**：Starlink一骑绝尘（10,000+颗），中国GW星座正在加速追赶，其他玩家差距巨大。

2. **中国卫星通信进入战略加速期**：20.3万颗频轨资源申报、GW星座200亿招标、2026年政府工作报告明确列为重点任务——政策意志前所未有。

3. **联通已具备先发优势**：A13-1牌照是运营商中除电信外的第二家，8省市已有落地。

4. **联通的最佳路径是"集成+应用"而非"自建"**：联通没有资金和基础设施自建卫星，但可以做卫星通信能力的整合者和分发者。

5. **5G-A/6G时代是卫星与地面网络融合的历史性窗口**：3GPP R17已建立NTN标准，R18/R19持续增强，高通X105芯片已在MWC 2026上发布，**5G手机直连卫星即将从演示走向商用**。联通现在布局NTN核心网改造和终端测试，恰逢其时。
